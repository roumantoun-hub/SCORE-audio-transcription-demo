# SCORE-audio-transcription-demo
This project is a web-based tool that allows users to upload audio files, select an instrument, and automatically generate sheet music and tablature without manual transcription. It also recommends musically similar pieces to help users learn and explore music within the same style.
## Sample Input and Output

### Sample Input
- [`Cat Jump.wav`](Vinzenz-Cat-Jump.wav)  
  A short audio clip used to demonstrate the transcription process.
